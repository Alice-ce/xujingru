## git的概念和基本使用
* **概念**：github是一个面向开源及私有软件项目的托管平台
* **目的**：借助github托管项目代码
* **基本概念**：
1. repository（仓库）
2. star（收藏）
3. fork（复制克隆项目）
4. pull request(发起请求)
5. watch(关注)
6. issue（事务卡片）
* **基本使用**：创建、编辑、修改了文件
